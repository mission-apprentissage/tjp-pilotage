Précisions importantes sur les accès

Il est possible de donner délégation à l’un des membres de son équipe pour l’accès à Orion.

Si l'on est rattaché à un établissement, on arrive directement dans Orion sur la page Panorama de l'établissement. On peut consulter les données de cette page et/ou utiliser le menu de navigation en haut de page pour explorer le reste du site.
