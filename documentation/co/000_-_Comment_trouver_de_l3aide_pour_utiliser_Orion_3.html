Glossaire

Il y a deux manières d’accéder au glossaire :

  • Il est possible à tout moment, depuis la navigation principale, d'ouvrir le volet glossaire sans quitter la page en cours ;

  • En cliquant sur les symboles dans les pages, on ouvre directement la définition correspondante du glossaire.

On y trouve :

  • Les définitions des termes et notions utilisés dans Orion ;

  • Des précisions sur les calculs des indicateurs (ex : taux d'emploi à 6 mois) ;

  • Des schémas explicatifs utilisables dans d'autres supports externes.
