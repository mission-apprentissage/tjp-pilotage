Actions avancées sur les données : console Formations

  1. Exporter
  2. Enregistrer la requête
  3. Requêtes favorites
  4. Désactiver une requête
  1. Exporter⚓

    Cette fonctionnalité permet d'exporter les données au format Excel ou CSV.

  2. Enregistrer la requête⚓

    Cette fonctionnalité permet d'enregistrer la combinaison de filtres en cours pour une visite ultérieure, tout en lui donnant un nom. Elle ne tient pas compte des éventuelles modifications d’affichage.

    Remarque : Nombre de requêtes favorites⚓

    Il est possible de sauvegarder autant de requêtes qu'on le souhaite.

  3. Requêtes favorites⚓

    On retrouve les requêtes sauvegardées dans la liste déroulante Requêtes favorites de la barre de filtres géographiques. Cliquer sur le nom de la requête permet de l’activer.

  4. Désactiver une requête⚓

    Cliquer sur Réinitialiser les filtres pour désactiver une requête active.
