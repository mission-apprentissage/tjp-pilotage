Cas de données manquantes

Il peut y avoir plusieurs explications aux cas de données manquantes dans la console. Les principales sont :

  • Données hors périmètre InserJeunes (si les taux d’emploi, de poursuite d’études, de devenir favorable, sont manquants) ;

  • Effectifs en apprentissage : ces effectifs ne sont pour l’instant pas fournis par la DEPP et donc non disponibles dans Orion (on affiche seulement si une formation d’un établissement propose la voie apprentissage ou non) ;

  • Formation trop récente, non éligible au continuum ;

  • Autres cas : pour tous les détails disponibles sur les données manquantes, consulter la section « Cas de données indisponibles ou manquantes » dans la FAQ.
