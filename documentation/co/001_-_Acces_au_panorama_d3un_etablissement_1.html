Navigation principale et barre de recherche

Le panorama Établissement est accessible depuis n’importe quelle page via le menu de navigation principal.

Depuis une autre page panorama, on peut également basculer entre les catégories par les onglets dédiés.

A l’arrivée sur le panorama établissement, il est proposé de rechercher l’établissement voulu, au choix :

  • Par nom ;

  • Par UAI ;

  • Par commune.

Remarque : Pré-sélection⚓

Selon le périmètre fonctionnel, la page peut être déjà pré-filtrée sur un établissement, mais il est possible de consulter n’importe quel autre établissement grâce à la barre de recherche qui restera visible en haut de page.
