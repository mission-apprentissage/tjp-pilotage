Rapprochement Oui/Non

Afin de réaliser le suivi de la transformation, il faut dans un premier temps rapprocher les demandes du constat de rentrée. Cette opération a lieu sur une campagne terminé lorsque le constat de rentrée est intégré dans Orion (généralement au mois de janvier qui suit la rentrée).

Exemple - Rentrée scolaire 2024⚓

Pour les données de la rentrée scolaire 2024, on s’intéresse aux demandes validées de la campagne 2023 pour la rentrée scolaire 2024.

Pour diverses raisons, il n’est pas possible de rapprocher toutes les demandes du constat de rentrée :

  • L’ouverture n’a pas eu lieu, on ne retrouve pas la formation dans l’établissement dans le constat de rentrée ;

  • La demande concerne une FCIL ou des effectifs en apprentissage uniquement et ces données ne sont pas disponibles dans le constat de rentrée ;

  • La demande n’a pas été saisie sur l’UAI ou le Code Diplôme référencé dans le constat de rentrée.

C’est notamment pour gérer ce dernier cas que l’indicateur « Rapprochement » a été créé dans la page Gestion des demandes.

Remarque - Accès à l’indicateur⚓

Cette fonctionnalité de gestion des rapprochements est accessible aux profils Admin RA uniquement
