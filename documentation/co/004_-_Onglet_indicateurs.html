Graphiques des indicateurs - Onglet « Indicateurs »

  1. Exporter les données
  2. Changer les indicateurs InserJeunes
  3. Changer les indicateurs offre/demande
  1. Exporter les données⚓

    Il est possible d'exporter les données de la formation telles que présentées dans les graphiques du volet pour le territoire choisi.

  2. Changer les indicateurs InserJeunes⚓

    Il est possible de changer les indicateurs des taux InserJeunes suivants :

    • Devenir favorable ;

    • Emploi à 6 mois ;

    • Poursuite d’études.

    Remarque : Affinage des données⚓

    Selon les indicateurs et la maille territoriale choisie, les données seront agrégées à un niveau plus ou moins fin. Par exemple, pour une académie ou un département, les taux InserJeunes resteront au niveau régional correspondant, mais les données d’effectifs pourront être plus affinées.

  3. Changer les indicateurs offre/demande⚓

    Il est possible de changer les indicateurs d'offre/demande suivants :

    • Taux de remplissage ;

    • Taux de pression.

    Remarque : Affinage des données⚓

    Selon les indicateurs et la maille territoriale choisie, les données seront agrégées à un niveau plus ou moins fin.
