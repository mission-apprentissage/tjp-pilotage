Créer un utilisateur

Pour créer un nouvel utilisateur dans Orion, il faut cliquer sur le bouton Ajouter un utilisateur et remplir le formulaire qui s’affiche (les champs obligatoires sont marqués d’une astérisque) :

  1. Email
  2. Prénom
  3. Nom
  4. Rôle
  5. Code région
  6. Fonction de l'utilisateur
  7. Envoyer
  1. Email⚓

    Ce champ permet de renseigner l'adresse email de l'utilisateur.

  2. Prénom⚓

    Ce champ permet de renseigner le prénom de l'utilisateur.

  3. Nom⚓

    Ce champ permet de renseigner le nom de famille de l'utilisateur.

  4. Rôle⚓

    On choisit ici quel rôle attribuer à l'utilisateur, en fonction des permissions que l'on souhaite lui donner dans l’outil.

  5. Code région⚓

    Ce champ apparaît seulement si on sélectionne un rôle en région académique dans le champ Rôle. Il s'agit du nom de la région. Il peut être définir par défaut.

    Retourner à l'écran
  6. Fonction de l'utilisateur⚓

    On peut choisir la fonction de l'utilisateur parmi les fonctions administratives listées.

    Retourner à l'écran
  7. Envoyer⚓

    Le bouton Envoyer permet de valider la saisie et de créer l'utilisateur.

    Retourner à l'écran
