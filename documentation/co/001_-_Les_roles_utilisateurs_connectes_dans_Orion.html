Les rôles utilisateurs « connectés » dans Orion

Dans Orion, une partie des données sur les formations et les établissements est publique et accessible à tous sans nécessité de se connecter.

→ Plus de détails sur les pages grand public dans Orion

En revanche, l’ensemble des fonctionnalités qui concernent les opérations et le pilotage de la transformation de la carte sont réservées aux utilisateurs connectés. Le périmètre d’accès varie selon le profil de ces utilisateurs.

Les sections ci-dessous détaillent les différents rôles utilisateurs, avec une explication brève de leurs permissions dans l’outil, selon leurs fonctions administratives et leurs attributions dans le cadre de la transformation. Pour mieux comprendre les rôles de ces différents acteurs dans la transformation de la carte, il est également possible de consulter la page du guide Comprendre le cycle annuel et les acteurs de la transformation de la carte.

Pour un aperçu exhaustif des permissions dans Orion par profil et par tâche, il est également possible de se référer à :

  • La page Rôles dans l’onglet « Admin » du site Orion (si le profil y donne accès) :

Remarque : Pages publiques⚓

Dans les sections « grand public » d’Orion (Panorama et Console), tous les utilisateurs ont accès aux mêmes données, avec ou sans accès connecté, et quel que soit leur profil. Sur ces pages publiques, l’accès connecté peut simplement modifier les filtres sélectionnés par défaut pour qu’il reflètent directement le périmètre pertinent pour l’utilisateur.

Exemple : le profil « Perdir » accèdera directement à la page panorama de son établissement, tandis que les profils « RA » verront directement leur région dans les consoles, etc.
